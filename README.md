# TT2 Automation Bot
A Python automation script for Tap Titans 2.

## 功能
- **自動領取每日簽到、寵物、寶箱、成就**
- **自動放技能、升級英雄**：啟動技能後根據抵達最大關卡所需時間來設定升級英雄次數
- **自動循環刷關**：達到最大關卡後自動蛻變，並且重複刷N次，解決每日需蛻變99次的活動

## 技術棧

### 開發與版控工具
- Visual Studio Code

### 後端技術
- Python
- pyautogui（滑鼠鍵盤控制）
- opencv（畫面辨識）
- keyboard / pynput（鍵盤監聽）
- time / schedule（定時排程）
